# Daniela Ahumada-C. research website

This is a simple static website. It can be edited with any text editor and hosted for free on GitHub Pages, Netlify, or Cloudflare Pages.

## Files

- `index.html`: all page sections and text.
- `styles.css`: colors, layout, typography, and responsive design.
- `script.js`: small footer year script.
- `assets/daniela-ahumada.jpeg`: profile photograph.

## How to update content

Open `index.html` and edit the section you need:

- `#about`: researcher profile.
- `#education`: academic training.
- `#research`: research lines.
- `#publications`: scientific publications.
- `#taxa`: described taxa.
- `#presentations`: selected conference presentations.
- `#recognition`: awards and distinctions.
- `#affiliations`: research groups and associations.
- `#profiles`: online profiles and pages featuring your work.
- `#contact`: contact email.

## Free publishing options

### GitHub Pages

1. Create a public GitHub repository.
2. Upload these files.
3. Go to Settings > Pages.
4. Select the main branch and root folder.
5. GitHub will give you a free public URL.

### Netlify

1. Go to Netlify.
2. Drag and drop the website folder.
3. Netlify will publish it and give you a free public URL.

## Notes

The `Described Species` section should only use figures from the original articles or public treatment pages. Each species entry should include the taxon authors.
